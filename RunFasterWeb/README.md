0| 工作路径
```shell
cd RunFaster/running
```

1| 安装依赖
```shell
pip3 install -r requirements.txt
playwright install
```

2| 启动Django
```shell
cd RunFaster

python manage.py runserver 127.0.0.1:9007
```

3| 进入后台
```浏览器访问
http://127.0.0.1:9007/index
```
